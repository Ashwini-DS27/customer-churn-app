# Customer Churn Prediction App

## 📌 Overview
This project predicts whether a customer will churn (leave) or stay using Machine Learning.

## 🚀 Features
- Customer churn prediction
- Interactive dashboard using Streamlit
- Data visualization (churn distribution, feature importance)
- Real-time user input prediction

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## 📊 Model
- Logistic Regression
- Accuracy: ~78%

## 🌐 Live App
[https://customer-churn-app-pswuzpefss98eljecjnren.streamlit.app/]

## 📁 Dataset
Telco Customer Churn Dataset

## 🎯 Key Insights
- Monthly charges influence churn
- Contract type plays major role
- Tenure reduces churn probability
